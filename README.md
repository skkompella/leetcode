# 🧩 My LeetCode Solutions

This repo auto-syncs my accepted LeetCode submissions once a day via a scheduled
GitHub Action, then regenerates this README with stats. Nothing runs on my machine.

**Not set up yet?** See [`SETUP.md`](SETUP.md), then trigger the workflow from the
**Actions** tab. After the first run this file is replaced with a full stats dashboard
(solved counts, difficulty breakdown, and a per-topic index linking to every solution).

Solutions land in [`solutions/`](solutions/) · README generated by
[`scripts/generate_stats.py`](scripts/generate_stats.py).
